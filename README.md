# Next.js + Directus Admin Starter

A minimal admin panel scaffold for Directus-backed products.

## Features
- Next.js (App Router) baseline
- Auth-ready structure (token-based)
- CRUD pages scaffold
- Clean UI baseline

## Use cases
- Headless CMS setups
- Internal dashboards
- Lightweight client admin panels

## Run
npm install
npm run dev
